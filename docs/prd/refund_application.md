# Формирование заявления на возврат

Автоматическое создание заявления на возврат в PDF по данным JSON согласно
`ТЗ_формирование_заявления_на_возврат_xhtml2pdf_Arial.md`.

## Использование

Установите зависимость:

```text
python -m pip install -r requirements.txt
```

Настройте пути в `config.ini`, затем запустите:

```text
python refund_application_generator.py "путь_к_файлу.json"
```

Скрипт принимает JSON в UTF-8/UTF-8 BOM или Windows-1251, проверяет данные,
заполняет xhtml2pdf-совместимый HTML-шаблон, сохраняет PDF и технический журнал
в настроенные папки, после чего открывает документ системной программой Windows.
Браузер и Microsoft Word не требуются. Для кириллицы используются системные
`C:\Windows\Fonts\arial.ttf` и `arialbd.ttf`; отдельный каталог шрифтов в проекте
и параметр `fonts_dir` не нужны. Единственный шаблон —
`templates/refund_application_2_xhtml2pdf.html`.

Длинная строка получателя выравнивается с отступом влево. Если после названия
организации в кавычках указано ФИО и вся строка не помещается, ФИО целиком
переносится на следующую строку после закрывающей кавычки.
